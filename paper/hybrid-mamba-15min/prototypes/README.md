# Prototype Pages

This folder keeps reusable HTML prototype pages for the paper deck.

- `method_flowchart.html`: method/process flowchart page
- `architecture.html`: main model architecture page
- `results_charts.html`: experiment/result charts page
- `base_ori_architecture.html`: baseline/original architecture reference
- `canvas_a.html`, `canvas_b.html`, `canvas_c.html`: canvas exploration drafts
