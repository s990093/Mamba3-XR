# TD-MoE：面向 MoE 的張量分解壓縮（ICLR 2026）

本資料夾整理論文 **「TD-MoE: Tensor Decomposition for MoE Models」**（Xu et al., ICLR 2026）的閱讀筆記與簡報素材，對應原始 PDF：`../5318_TD_MoE_Tensor_Decompositi.pdf`。

官方程式碼：<https://github.com/ust-xu/TD-MoE>

---

## 1. 這篇在講什麼（一句話）

把同一層 **所有專家的權重疊成三維張量**，用 **Tucker 分解** 做 **跨專家、聯合** 的低秩近似；並用 **多線性白化** 與 **三維秩分配** 讓壓縮在固定參數預算下更貼近真實資料幾何。

---

## 2. 動機與問題

- **MoE** 透過路由只啟用部分專家，擴展能力強，但 **專家權重總量** 造成巨大 **記憶體** 壓力。
- 既有 **矩陣分解 / SVD** 類方法多在 **「逐專家」** 粒度上壓縮，**忽略專家之間共用的結構冗餘**。
- 高壓縮比時，逐專家分解容易出現 **精度掉得快** 的問題。

---

## 3. 方法總覽（三步驟）

| 元件 | 作用 |
|------|------|
| **跨專家張量化（Cross-expert tensorization）** | 將 \(K\) 個專家權重 \(W^{(i)}\in\mathbb{R}^{d_{\text{out}}\times d_{\text{in}}}\) 疊成 \(T\in\mathbb{R}^{K\times d_{\text{out}}\times d_{\text{in}}}\)，讓「專家維」成為可分解的一維。 |
| **多線性白化（Multi-linear whitening）** | 用校正集上的 **輸入共變異** 與 **輸出梯度共變異** 對張量沿 mode 做白化，降低特徵相關造成的病態，使後續 Tucker 更穩、更貼資料。 |
| **三維秩分配（3D rank allocation）** | 在目標壓縮比 \(\rho^\*\) 下，搜尋 \((r_1,r_2,r_3)\)，使 Tucker 後參數量 **逼近預算**（論文給出由 \((r_1,r_2)\) 推 \(r_3\) 的閉式關係）。 |

**推論時**：白化逆變換 **吸收進因子矩陣**（re-coloring），不增加額外推理開銷。

---

## 4. 核心數學（精簡版）

**（1）校正目標（保留啟用行為）**

在校正分佈 \(\mathcal{D}_{\text{calib}}\) 上最小化專家線性層的輸出誤差：

\[
\min_{\{\hat W^{(i)}\}} \mathbb{E}_{x\sim \mathcal{D}_{\text{calib}}}\left[\sum_{i=1}^{K}\|W^{(i)}x-\hat W^{(i)}x\|_2^2\right].
\]

**（2）張量化**

\[
T \in \mathbb{R}^{K\times d_{\text{out}}\times d_{\text{in}}}.
\]

**（3）Tucker 分解（寫在白化後張量 \(T_w\) 上）**

\[
T_w \approx G \times_1 U_1 \times_2 U_2 \times_3 U_3,
\]

其中 \(G\in\mathbb{R}^{r_1\times r_2\times r_3}\)，\(U_1\in\mathbb{R}^{K\times r_1}\)、\(U_2\in\mathbb{R}^{d_{\text{out}}\times r_2}\)、\(U_3\in\mathbb{R}^{d_{\text{in}}\times r_3}\)。

**（4）多線性白化（論文式 (4)）**

\[
T_w = T \times_2 S_{\text{out}} \times_3 S_{\text{in}},
\]

\(S_{\text{in}}, S_{\text{out}}\) 由共變異矩陣的正則化平方根逆得到。

**（5）參數量與壓縮**

- 原始：\(P_{\text{orig}} = K\,d_{\text{out}}\,d_{\text{in}}\)。
- Tucker：\(P_{\text{tucker}} = r_1r_2r_3 + (Kr_1 + d_{\text{out}}r_2 + d_{\text{in}}r_3)\)。

在固定壓縮比下，可由 \((r_1,r_2)\) **閉式解出** 對應的 \(r_3\)（論文式 (5)），再投影到合法整數區間，用二維掃描完成秩搜尋。

---

## 5. 和誰比、差在哪

| 方向 | 代表思路 | TD-MoE 的差異 |
|------|-----------|----------------|
| **逐專家 SVD**（如 MoE-SVD） | 每個 \(W^{(i)}\) 各自低秩 | **聯合** 分解，顯式壓縮 **專家維** 冗餘（\(U_1\) 可視為「元專家」子空間）。 |
| **剪枝 / 合併** | 刪專家或合併相似專家 | 不需硬決策保留誰；以 **連續** 的低秩表示取代離散刪併。 |
| **僅共享輸入投影** 類假設 | 冗餘集中在輸入側 | TD-MoE 用 **三模 Tucker + 白化**，較不依賴單側強假設。 |

## 6. 實驗與主要結果（論文敘述摘要）

- **模型**：Qwen2-57B-A14B、Mixtral-8×7B、Phi-3.5-MoE 等。
- **任務**：七項常識推理（zero-shot）+ WikiText-2 / PTB / C4 困惑度。
- **設定**：256 筆 WikiText-2 作校正；**嚴格 post-training、無微調**。
- **亮點（論文聲稱）**：
  - **約 20% 壓縮**：效能 **接近無損**。
  - **40% / 60% 壓縮**：相對 MoE-SVD、NAEE 等分解類基線，可達 **約 11%–14% 量級**的相對改進（依模型與設定而異）。
- **消融**：聯合張量分解、白化、秩分配 **各自有貢獻**。
